1. 글꼴 속성
font-size   -글자의 사이즈
            고정 사이즈
            -px(고정 화소 기준 사이즈)
            -pt(고정 사이즈)

            상대사이즈
            -em 부모 기준 요소의 사이즈를 기준으로 하는 사이즈
                부모 13px
                자식 1em -> 자식 13px
                     1.2em -> 자식 15.6px
            -rem 최상의 태그 기준 사이즈
            html 13px
            1rem ->13px
            1.2rem-> 15.6px
             ex) 최상위픽셀 13
            인데 rem으로 30px을 만들고 싶을때
            30/13 =2.307692307692308 //2.31 정도로 대충 맞추면됨

            폰트사이즈 바꾸고 싶을때,
            상대사이즈를 쓰면 최상위 사이즈만 바꾸면 되니까 편리함.
            그래서 보통 em보다 rem을 많이 씀.

2.공간
    display - 공간 속성을 변경
              block - 공간
                    - 너비,높이
                    - 상하좌우 여백(padding, margin)

              inline - 글의 속성
                     - 너비, 높이 X
                     - 모든 방향(상하좌우) padding 가능
                     - 좌우 여백 margin 가능
              inline-block
                     - inline과 block을 절충
                     - 줄개행 X
                     - 너비 높이O, 모든 방향 margin 가능
              none - 감춤처리
              표와 같이 만들어주는 속성 값;
              table - 테이블 행
              table-cell - 테이블 열

3.레이아웃
    box-sizing
        1)내용 영역 기준
        content-box(기본값)
        내용 영역의 사이즈를 유지하면서 확대
        총 사이즈 = 내용 영역 + 내부여백(padding) + 경계

        2)경계 영역 기준
        border-box (강사님 선호)
        경계를 기준으로 사이즈를 정하는 방식
        경계 + 내부여백(padding)에 따라서 내용 영역의 사이즈가 변경 된다.

6.포지션
    - 배치
    position
        -stastic(기본값)
        
        상대위치
        -relative
            -기준(요소 자신)
            -자기 자신이 위치한 위치 기준으로 상대적으로 이동(자기 자신이 이동)
        -absolute
            -기준 -상대위치로 지정한 부모 또는 조상
                -부모,조상 중 누구라도 상대위치 속성을 가진다면(relative,absolute,fixed)
                기준내에서 상대적으로 이동
                상대속을 가지고 있는 부모가 없다면 전체문서 기준 (html)
                
                
        -fixed
        -기준 보이는 화면 기준으로 상대적으로 이동

        상대위치는 상대적으로 이동하기 위한 것이므로 하기 추가 속성 존재
        
        top 위에서 아래로
        reight 오른쪽에서 왼쪽으로
        bottom 아래에서 위로
        left 왼쪽에서 오른쪽으로
        z-index 각 요소의 층위
                숫자가 클수록 앞쪽에 있고
                숫자가 작아질수록 뒤쪽에 있다
                예) z-index: 10, z-index:9;
                 10이 9보다는 앞에 있다
