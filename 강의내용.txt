*선택자
-* 전체 태그 선택자

-태그 선택자
(태그명으로 선택)

-class
.클래스명

-id
#아이디명

*태그의 style 속성으로 정의 하는 방법

*적용범위
태그 > class >id > 태그의 style 속성으로 정의

*css 적용 우선 순위
(적용범위가 좁을 수록 우선 순위가 높다)
태그의 속성 style > id > class > 태그

*우선순위 강제 적용(좋은 방법X 꼬임)
!important -> 우선순위 강제 적용

*조상 자손 선택자
왼쪽 -> 오른쪽 (왼쪽으로 갈수록 상위 태그, 오른쪽으로 갈수록 하위 태그)

*부모 자식 선택자
바로 위 층위, 바로 아래 층위
부모 > 자식

*그룹 선택자
콤마(,)로 구분하여 여러 선택자를 적용

*가상 클래스 선택자
선택자:상태값
-요소의 상태로 선택
예1) 체크박스, 라디오 박스 -> 체크가 되었는지?
선택자: checked

예2) 요소에 마우스가 올려 있는 상태인지?
선택자:hover

예3) 요소가 읽기 전용 상태인가?
선택자:read-only

예4) 선택자가 아닌 상태인가?
선택자1에 선택자2가 포함 되어 있지 않은 상태인가?
선택자1:not("선택자2")

예5) 첫번째 자식인가?
선택자:first-child

예6) 마지막 자식인가?
선택자:last-child

예7) 몇번째 자식인가?
선택자:nth-child
nth-child(숫자) ->번째 요소
nth-child(수식) ->수식 계산 후 선택
2n ->2,4,6,8,10......
(n은 1부터 시작)
2n+1 -> 1,3,6,9,11....
(뒤에 수식이 추가 되어 있으면 n은 0부터 시작)
-가상으로 추가하여 선택하는 선택자

선택자:after
-선택자의 자손으로 가장 끝에 추가

선택자:before
-선택자의 자손으로 가장 앞에 추가

*결합자 (가운데 기호)
부모 > 자식

형제

인접형제(+)
선택자1+선택자2 -> 선택자1(왼쪽위치)을 인접 형제로 가진 선택자2를 선택
ex) .menu1(왼쪽위치) ~ .menu2 
일반형제(~)
선택자1~선택자2 -> 선택자1을 형제(왼쪽위치)로 가지고 있는 선택자2를 선택(인접X)

속성선택자
선택자[속성명="속성값"] {}
